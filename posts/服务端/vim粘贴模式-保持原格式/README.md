## 背景

在使用 `vim` 编辑时，大批量的外部文本粘贴，可能无法保持原格式，而造成持续的文本缩进。

然而 `vim` 提供了 `粘贴模式` , 具体如下：

## 使用

在 `vim` 在命令模式中输入

```sh
:set paste
```

后, 再按 `i` 进行编辑，这时候粘贴大文本时，将保持原文件格式。

> 退出 粘贴模式 输入 `:set nopaste`
